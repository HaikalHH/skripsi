# Reporting

Domain report dan analytics transaksi.

- `transaction-summary/`: agregasi transaksi dasar per periode.
- `report-builder/`: parsing query report natural language dan pembuatan jawaban analytics.
- `monthly-pdf/`: payload dan attachment PDF laporan bulanan.
- `query-language/`: kumpulan phrase/term untuk mengenali intent report.
- `shared/`: tipe internal reporting.
